Back To Dirt Plugin.

 - Back To Dirt is a bukkit plugin that allows you to transform a path into a dirt again.
 - I developed this plugin after rebuilding my hardcore world village, so I suffered a lot breaking these blocks.
 - Feel free to use this plugin wherever you want!

How to use?
 - Just hold shift and right click on a dirt path and it will turn into dirt.
